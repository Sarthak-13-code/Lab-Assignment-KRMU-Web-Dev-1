JavaScript Quiz Game

This is a simple JavaScript quiz game that runs entirely inside the browser console. It asks the user multiple questions using prompt(), checks the answers, gives instant feedback using alert(), and shows the final score at the end.

This project was created as part of Lab Assignment 4, focusing on arrays, loops, conditionals, functions, and basic input/output in JavaScript.
